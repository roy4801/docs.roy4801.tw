# Use SMTP send gmail

- `smtp.gmail.com`
- username/password: `your@gmail.com/password`
- port: 587 TLS, 465 SSL

!!!warning
	Remember to turn on "Allow less secure apps" in <https://myaccount.google.com/security#connectedapps>
