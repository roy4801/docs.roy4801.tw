# vscode
## keymap
### Windows
* indent
	* left `ctrl+{`
	* right `ctrl+}`

* fold/expand block
	* fold `ctrl+shift+{`
	* expand `ctrl+shift+}`

### macos

* command ⌘
* option ⌥
* control ⌃
* shift ⇧

* indent
	* left `cmd+{`
	* right `cmd+}`

* fold/expand block
	* fold `cmd+opt+{`
	* expand `cmd+opt+}`

## Other

- List extensions from CLI

```bash
code --list-extensions
```
